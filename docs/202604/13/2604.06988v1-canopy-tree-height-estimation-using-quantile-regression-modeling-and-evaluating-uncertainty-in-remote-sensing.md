## 摘要
准确的树高估计对于生态监测和生物量评估至关重要。我们将分位数回归应用于现有基于卫星数据的树高估计模型，以引入不确定性量化。目前大多数树高估计方法依赖于点预测，这限制了它们在对风险敏感的场景中的适用性。在这项工作中，我们表明，只需对给定的预测头进行微小修改，现有模型就可以通过分位数回归进行调整，以提供经过统计校准的不确定性估计。此外，我们展示了我们的结果如何与遥感中已知的挑战（例如，地形复杂性、植被异质性）相关联，表明模型在更具挑战性的条件下置信度较低。

## Abstract
Accurate tree height estimation is vital for ecological monitoring and biomass assessment. We apply quantile regression to existing tree height estimation models based on satellite data to incorporate uncertainty quantification. Most current approaches for tree height estimation rely on point predictions, which limits their applicability in risk-sensitive scenarios. In this work, we show that, with minor modifications of a given prediction head, existing models can be adapted to provide statistically calibrated uncertainty estimates via quantile regression. Furthermore, we demonstrate how our results correlate with known challenges in remote sensing (e.g., terrain complexity, vegetation heterogeneity), indicating that the model is less confident in more challenging conditions.

### 6. 论文的主要结论与发现
*   分位数回归能够有效地为现有树高模型提供校准良好的不确定性估计，且计算成本远低于集成方法。
*   相比于 Lang 等人的高斯回归模型，本文方法生成的预测区间更窄（MPIW 更小），但实际覆盖率（PICP）更高。

*   **困难场景的高不确定性**：模型在森林边缘和陡峭地形（坡度较大）区域表现出明显更高的不确定性，这与遥感领域已知的 GEDI 定位误差和地形噪声挑战高度吻合。

### 7. 优点
*   **轻量级与易用性**：只需对现有预训练模型添加一个预测头并进行少量微调（2个 Epoch），无需从头训练庞大的集成模型。
*   **无需分布假设**：分位数回归不需要假设预测误差服从特定分布（如高斯分布），能更好地拟合真实世界中不对称的树高分布。
*   **结合领域知识**：引入 Shift-Resilient Loss 有效缓解了遥感数据特有的地理定位偏差问题。

### 8. 不足与局限
*   **极端值覆盖不足**：模型倾向于高估极矮的植被（<5m），同时严重低估极高的树木（>30m），在这些区间的经验覆盖率偏离理想值。
*   **标签噪声干扰**：部分高置信度的错误预测实际上是由 GEDI 标签本身的错误引起的（如农田被标记为 30m 高树），目前模型还无法自动清洗这些噪声。
*   **评估范围限制**：实验主要在一个包含 250 个样本的测试集上进行，尚未展示在全球不同气候带和森林类型下的全面泛化能力。
